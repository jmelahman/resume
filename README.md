Personal resume in LaTeX and Open Document formats. The cover letter is written in a plain text document to allow for easy changes as needed. Necessary, user-specific changes in /src/rusume.tex are prefaced with comments of the format '### <Information to add>'.

To make the pdfs using LaTeX, run `bash make.sh` from the parent directory. 

The open document file has a lot of peculiar spacing and indenting from trying to fit all information onto the page which is why I decided to re-do it in a LaTeX to have absolute control over formatting.
